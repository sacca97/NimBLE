Playing with NimBLE and the nRF52840 board
==========================================

**NimBLE** is the world's first open-source BLE stack, it is compliant to Bluetooth 5.0 and supports Bluetooth Mesh.

Components
----------

* Controller
  
  * Physical layer
  * Link layer

* Host
  
  * L2CAP
  * SecurityManager (SMP)
  * (G)ATT
  * GAP
  * HCI
